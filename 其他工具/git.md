# 常用linux命令

```
cd	跳转目录
	cd ..	返回上一个目录
	cd		进入默认目录
	cd desktop	跳转到desktop目录
	
pwd	显示当前目录路径
clear 清屏
ls	列出目录下的文件（ll包含更多信息）

touch 新建一个文件
rm	删除一个文件

mkdir （）	新建文件夹
rm -r ()	删除文件夹

mv (目标对象) (目录) 移动文件

history	查看历史命令
```
